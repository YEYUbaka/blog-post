在软件开发的世界里，Git 是几乎每个程序员的必备技能。无论你是写代码、写文档，还是做网页，Git 都能帮你管理修改记录，防止“代码丢失”的惨剧，同时，他还解决了多人协作会带来的问题，允许成员并行作业，而不影响主要代码库。
:::

### 1. 什么是 Git？

简单来说，Git 是一个分布式版本控制系统。

- 版本控制：能记录文件的每一次改动。写错了？没关系，一键回退到昨天的版本。

- 分布式：每个人的电脑上都有完整的版本库，断网也能工作，同时保证了多个成员的并行作业。

### 2. 安装与配置
首先，你需要下载安装 Git（官网下载（https://git-scm.com/downloads））。安装完成后，打开终端（Windows 下是 Git Bash，Mac/Linux 是 Terminal）。

这是最重要的第一步：自报家门。 因为 Git 是用来协作的，所以必须知道“是谁提交了代码”。
```bash
git
```
