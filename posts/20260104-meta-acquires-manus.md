:::note{type="import"}
2025年12月29日，Meta Platforms 宣布以超过 20 亿美元收购 Manus——一家总部位于新加坡、由中国团队创立的通用 AI Agent 公司。这是 2025 年 AI 领域金额最高的并购案，也是"AI Agent"从一个技术概念转变为科技巨头核心战略资产的标志性事件。
:::

## Manus 是什么？为什么值 20 亿美元？


<!-- REAL_PHOTO: Manus AI 产品界面截图或官方宣传图 -->
Manus 是 Butterfly Effect 公司（一家在中国创立、总部设在新加坡的 AI 初创公司）在 2025年3月6日发布的产品。它的核心定位是**通用 AI Agent**——你给它一个目标，它自主规划、搜索、分析、并交付结果。

与 ChatGPT 或 Claude 这类"对话式 AI"不同，Manus 不需要用户一步步指导。用户说"帮我做一份特斯拉和比亚迪的对比研究"，Manus 会自主搜索两家公司的最新财报、行业报告、技术分析，整理成一份结构化的对比文档，然后交付给你。

Manus 发布时引起了现象级的关注——其邀请码在中国社交媒体上被炒到数万元人民币。虽然批评者指出 Manus 底层调用的是现有大模型（包括 Claude 和 GPT），其核心价值在于 Agent 工作流的编排而非模型本身，但这种"AI Agent 操作系统"的定位恰恰是 Meta 看重的。

到被收购时，Manus 已经积累了数百万用户，其中相当比例的付费用户，证明了**用户愿意为"AI 帮我做事"付费**——而不只是"AI 跟我聊天"。

## 为什么是 Meta？


<!-- AI_GEN: Meta收购Manus的战略逻辑图：社交+Agent+开源三大支柱 -->
Meta 收购 Manus 的逻辑有三个层次。

**第一，补 Agent 短板。** Meta 在开源大模型领域（Llama 系列）已经建立了强势地位，但在 AI Agent 的产品化方面明显落后于 OpenAI（GPT-5 原生 Agent）、Anthropic（Claude Computer Use）和 Google（Gemini Agent）。收购 Manus 是最快的方式把 Agent 产品能力补齐。

**第二，社交+Agent 的想象空间。** Meta 拥有 Facebook、Instagram、WhatsApp 三大社交平台，月活用户超过 30 亿。将 Manus 的 Agent 能力整合到这些平台中——比如在 WhatsApp 里帮你自动处理行程预订、在 Instagram 里帮你管理品牌合作——可以创造全新的用户价值和广告场景。

**第三，对冲 OpenAI。** Meta 和 OpenAI 的关系已经恶化到几乎公开敌对的程度。Zuckerberg 多次在公开场合抨击 OpenAI 的"假开源"策略。收购 Manus 让 Meta 有了独立的 Agent 能力，减少了对 OpenAI 生态的依赖。

## 地缘政治的暗流

这笔交易最敏感的部分在于**Manus 的中国背景**。

Butterfly Effect 的创始团队来自中国，公司核心技术研发也在中国进行（虽然法律总部在新加坡）。在美国对华 AI 技术转移高度敏感的 2025-2026 年时间节点上，一家美国科技巨头收购一家有中国背景的 AI 公司，必然会引发监管审查。

CNN 和 BBC 的报道均提到，CFIUS（美国外国投资委员会）可能会审查这笔交易。如果审查不通过，交易可能被叫停——这在 AI 领域已有先例（2024年，美国曾阻止一家中资背景的基金收购美国 AI 芯片公司的少数股权）。

在中国方面，Manus 作为一家"走出去"的中国 AI 团队，其核心技术被美国公司收购，也可能引发中国监管层的关注。尤其是如果 Manus 的技术涉及数据安全或国家安全领域。

## 对中国 AI 创业生态的启示

Meta 收购 Manus 对中国 AI 创业生态有着复杂的影响。

积极的方面：它证明了中国 AI 团队在全球市场上的竞争力。Manus 不是靠"抄袭硅谷"成功的——它的 Agent 产品理念和用户体验设计都有自己的原创性。这笔交易也为其他中国 AI 创业者提供了一个标杆——做出真正有价值的 AI 产品，就有被全球科技巨头认可的可能。

隐忧的方面：又一个优秀的中国 AI 公司被美国买走了。这让人联想到AI人才和技术的"流失"问题——当中国最优秀的 AI 创业者选择在新加坡注册公司、最终被美国收购时，对于中国 AI 生态的长期发展意味着什么？

## 2026年的 Agent 竞赛格局

Meta 收购 Manus 之后，全球 AI Agent 的竞争格局变得更加清晰：

- **OpenAI**：GPT-5 原生 Agent + Operator（桌面操作）+ ChatGPT 的 3 亿+ 用户基础
- **Google**：Gemini 3 Agent + Google Workspace 生态 + Android 设备触达
- **Meta**：Llama 4 开源 + Manus Agent 产品 + 30 亿社交用户
- **Anthropic**：Claude Computer Use + 多 Agent 协作（Agent Teams）+ 企业安全优势
- **Microsoft**：Copilot + Office 生态 + GitHub Copilot Agent Mode

而对于国内的字节、阿里、腾讯、百度来说，Manus 被收购意味着：**Agent 赛道已经进入大平台整合阶段**，独立的 Agent 创业公司将面临更大的竞争压力。

## 结语

Meta 收购 Manus 是一笔"定义性的交易"（defining deal）——它定义了 AI Agent 从一个有趣的实验方向变成了科技巨头的必争之地，也定义了 2026 年 AI 竞争的核心战场将从"模型层"转向"应用层"。

Mark Zuckerberg 在收购后的内部备忘录中写道：「AI 的下一个十亿用户不会来自更好的聊天机器人，而是来自能真正帮人做事的 Agent。」这句话可能准确地描述了 AI 行业 2026 年的主旋律。
